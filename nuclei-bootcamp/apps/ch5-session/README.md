# Challenge 5 (CH5)

## Overview
This challenge involves a server-hosted application available at the given URL.
Participants do not receive source code or internal details.

---

## Scope
- Challenge ID: CH5
- Access type: Remote
- Testing model: Black-box

---

---

## Submission
